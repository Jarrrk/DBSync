DBSync

A Linux commandline utility for pushing database changes with codebase changes

View on GitHub Download .zip Download .tar.gz

What is DBSync?

DBSync is a Linux commandline utility allowing for easy database changes between application versions.

Usage

Install Instructions

Simply move dbsync from the downloaded archive in to your /usr/bin/ folder.
